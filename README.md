# Authorized-User-Post-Management-Backend
A backend focused web application built with Express.js that implements secure user authentication and post management using JWT (JSON Web Token) and bcrypt. The project demonstrates how authentication, authorization and relational data handling work together in a real world backend system.

## ✨ Features
- User registration with hashed passwords  
- Login system with JWT stored in cookies  
- Protected routes using middleware  
- Create, edit and update posts  
- Like/unlike functionality  
- User–Post relationship (MongoDB references)  
- Logout with cookie invalidation  

---

## 🧠 Key Concepts Implemented
- Authentication vs Authorization  
- Middleware based route protection  
- Password hashing & salting  
- Token based session management  
- MongoDB population (`.populate()`)  
- RESTful route structuring  

---

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB + Mongoose  
- **Authentication:** JWT (JSON Web Token), bcrypt  
- **Templating:** EJS
